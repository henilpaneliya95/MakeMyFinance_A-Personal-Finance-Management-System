# 💸 MakeMy Finance

**MakeMy Finance** is a full-stack AI-powered personal finance management application that helps users make smarter money decisions. The app enables users to track expenses, income, savings, financial goals, and receive smart AI-based financial insights.

---

## 📁 Project Structure

MakeMy Finance/
├── .venv/ # Python virtual environment (ignored)
├── frontend/ # React.js (or Vite) frontend with Tailwind CSS & Shadcn/ui
├── backend/ # Python backend (Django or FastAPI)

---

## ⚙️ Features

- 🔐 User Authentication (Login / Signup)
- 📊 Dashboard with real-time analytics
- 📉 Expense & Income tracking
- 🎯 Financial goals and savings tracker
- 🤖 AI-based financial recommendations
- 🎙️ Voice command input (Mic Logger)
- 📈 Expense breakdown, transactions, heatmaps
- 🧾 Review & budgeting tools
- 🪪 Secure and user-friendly design

---

## 🛠 Tech Stack

### 🌐 Frontend

- React.js / Vite
- Tailwind CSS
- Shadcn/ui
- Lucide Icons

### 🧠 Backend

- Django or FastAPI
- RESTful API
- JWT Authentication

### 🗃️ Database

- SQLite / PostgreSQL (as configured)
- Axios for frontend-backend communication
